数组去重

```javascript
var arr = [1, 2, 3, 4, 4, 3, 2, 1];
Array.prototype.removeDuplicate = function () {
  var n = [];
  for (var i = 0; i < this.length; i++) {
    if (n.indexOf(this[i]) == -1) {
      n.push(this[i])
    }
  }
  return n;
}
console.log(arr.removeDuplicate());
```
